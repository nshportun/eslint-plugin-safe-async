# Support

## How to get help

This is a solo-maintained open-source project. Support is provided on a
best-effort basis.

### For questions and general discussion

Open a [GitHub Discussion](https://github.com/nshportun/eslint-plugin-safe-async/discussions).
This is the preferred place for questions, configuration help, and feedback.

### For bugs and false positives

Open a [GitHub Issue](https://github.com/nshportun/eslint-plugin-safe-async/issues)
using the appropriate template.

Please search existing issues before opening a new one.

### For security vulnerabilities

See [SECURITY.md](./SECURITY.md).
Do not report security issues in public issues or discussions.

## What is not supported

- Feature requests that significantly expand scope
- Compatibility with ESLint versions below 9.0.0
- Support for legacy `.eslintrc` config format beyond what is documented

## Response time

Issues and discussions are checked regularly but response time is not
guaranteed. Critical bugs are prioritized. Feature requests may take
longer or may be declined.
